# 10x10 贪吃蛇

一个无需安装依赖的静态网页小游戏。

## 运行

直接用浏览器打开：

```text
D:\.codex\snake-10x10\index.html
```

## 公网部署

这个项目可以直接部署到 GitHub Pages、Netlify 或 Vercel 等静态托管平台。部署后，把生成的网址发给别人，对方即可在浏览器中游玩。

详细步骤见：

```text
D:\.codex\snake-10x10\DEPLOY.md
```

## 规则

- 游戏区域是 10x10 格子。
- 小蛇初始长度为 2，出生在地图中心。
- 点击“开始游戏”后开局。
- WASD 控制方向：W 上、A 左、S 下、D 右。
- 蛇头碰到苹果后吃掉苹果，身体长度增加 1。
- 可选择 20 个苹果、50 个苹果、无限模式三种目标。
- 选择无限模式时，页面右侧会显示本机浏览器保存的历史最佳纪录。
- 页面右上角可在中文和英文之间切换。
- 游戏初始速度较慢，吃到的苹果越多，小蛇移动越快。
- 撞墙或撞到自己会结束游戏。
